* cl-lns
  Basic interface for [[http://ln-s.net][ln-s.net]] in Common Lisp.

** Basic usage
   #+BEGIN_SRC lisp
   (ql:quickload :cl-lns)

   (cl-lns:lnsify "http://google.com")
   #+END_SRC
